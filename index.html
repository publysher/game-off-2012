Dinner at the Long Branch Saloon

A classic text adventure (interactive fiction), placed in Dodge Town, Kansas in 1878.

Playing

The game can be played in a browser on my website

Prerequisites

  • Download and install the lates version of Inform
  • Download and install the Undo Output Control extension
  • Download and install the Vorple extensions

Building

  • Open the Long Branch.inform in Inform.
  • Build a release
  • Happy playing :)
